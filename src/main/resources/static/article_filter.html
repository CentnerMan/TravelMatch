page_number номер страницы, отсчет с 1. Значение по умолчанию 1:

page_size количество элементов на странице, значение по умолчанию = 20:

order_direction направление сортировки, возможные значения ASC и DESC. Значение по умолчанию ASC, регистронезависимая запись:

order_by_properties список полей для сортировки. Разрешенные значения: id,title,text,category,author,created,lastUpdated,city,language. Можно перечислить несколько через запятую:

id статьи, точное соответствие:

author_id id автора статьи, точное соответствие:

category_id id категории статьи, точное соотвествие:

city_id id города, точное соответствие:

language_id id языка народов мира, точное соответствие:

text, поиск по вхождению строки в текст статьи, регистронезависимый поиск:

title_equal название статьи, точное соответствие:

title_contains поиск по вхождению строки в заголовок статьи:

created_equal дата создания, точное соответствие с учетом секунд, пример 2020-03-03T00:00:00:

created_before дата создания, поиск статей, созданных до указанной даты, пример 2020-03-03T00:00:00:

created_after дата создания, поиск статей, созданных после указанной даты, пример 2020-03-03T00:00:00:

updated_equal дата обновления статьи, точное соответствие с учетом секунд, пример 2020-03-03T00:00:00:

updated_before дата обновления, поиск статей, обновленных до указанной даты, пример 2020-03-03T00:00:00:

updated_after дата обновления, поиск статей, обновленных после указанной даты, пример 2020-03-03T00:00:00:

likes_equal количество лайков на точное равенство:

likes_greaterOrEqual количество лайков, большее либо равное указанному значению:

likes_lessOrEqual количество лайков, меньшее либо равное указанному значению:

dislikes_equal количество дизлайков на точное равенство:

dislikes_greaterOrEqual количество дизлайков, большее либо равное указанному значению:

dislikes_lessOrEqual количество дизлайков, меньшее либо равное указанному значению:

rating_value_count_equal количество оценок для рейтинга на точное равенство:

rating_value_count_greaterOrEqual количество оценок для рейтинга, большее либо равное указанному значению:

rating_value_count_lessOrEqual количество оценок для рейтинга, меньшее либо равное указанному значению:

rating_equal средний рейтинг статьи на точное равенство:

rating_greaterOrEqual средний рейтинг статьи, больший либо равный указанному значению:

rating_lessOrEqual средний рейтинг статьи, меньший либо равный указанному значению:
